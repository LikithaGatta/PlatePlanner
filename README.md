# PlatePlanner

Plate Planner was developed to help create a healthier society through a web app that allows users to track, create, and share recipes and their progress.
clone the repo

git clone https://github.com/SanjVal23/PlatePlanner.git


cd PlatePlanner


switch branch

git checkout recovery


Open 2 terminals

terminal 1:

cd backend -> npm run dev


Install dependencies in terminal 2

npm uninstall typescript

npm install --save-dev typescript@4.9.5

npm install --legacy-peer-deps

npm install react-router-dom@7.9.6

npm install dayjs


terminal 2:

cd my-app -> npm start
